12.1 Overview
untitled-d7d990e6-64fb-4fcd-9965-835a44ab4281.svg Web Search, Feature Dependencies and Multimodal Learning

Web search refers to the process of retrieving and ranking web pages according to their relevance to a user’s query. This functionality is typically supported by search engines such as Google and Yahoo. Traditional search engines are built upon information retrieval (IR) techniques to identify relevant documents. However, unlike classical IR settings, the web introduces a distinctive characteristic: hyperlinks that connect web pages, forming a large-scale directed network. This structural property has significantly extended traditional IR research and led to new directions, including link analysis, search engine architecture, web log analysis, online communities, and recommender systems.

The web contains an enormous volume of data in diverse formats, making it an ideal domain for machine learning methods. In particular, unsupervised learning plays an important role, as it can discover hidden patterns (e.g., user browsing behavior in large log files), reveal structural relationships when the web is modeled as a graph, segment users into meaningful groups, and detect anomalies, all without requiring labeled data.

In text-based retrieval, the bag-of-words representation has historically been highly effective in information retrieval tasks, often outperforming more complex textual representations. This model assumes independence between words, ignoring syntactic and semantic relationships. However, many modern applications require richer representations and ranking functions that integrate multiple feature sets. While bag-of-words features remain important, additional sources of information, such as linguistic features, document structure, hyperlinks, metadata, and non-textual content, can substantially improve performance.

From a machine learning perspective, these heterogeneous feature sets are often interpreted as multi-view data or multimodal data. Modern approaches, such as attention mechanisms, allow models to dynamically weigh the importance of different features by learning dependencies among them. In addition, contrastive learning has become an effective framework for learning joint representations across modalities (e.g., text and images), enabling models to better capture similarities and differences in a shared embedding space.

Week 12 lecture covers web search using Google’s search engine architecture, followed by link analysis and the PageRank algorithm. It then introduces attention mechanisms for modeling feature dependencies, discusses multi-view data and linear feature-based models, and concludes with multimodal learning, fusion strategies, and contrastive learning techniques.
